Quizzo is a multiplayer quiz game inspired by Kahoot! where teams compete by answering questions correctly. Each correct answer pulls the rope toward the team’s side. The team that pulls the rope to their side wins the game.

Feature:

-Multiplayer quiz gameplay

-Tug-of-war progress bar mechanic

-Real-time scoring

-Multiple choice questions

-Classroom-friendly design

Technologies Used:

-HTML

-CSS

-JavaScript

-Firebase Database

How to Run the Project:

-Download or clone the repository from GitHub

-Open the project folder

-Run index.html in your browser

-Start the quiz game

-Or access this link to start the quiz game: https://quizzo-a7d72.web.app/
